# IDog

An IDWall challenge

[https://idog-app.herokuapp.com/](https://idog-app.herokuapp.com/) file for details

### Install dependencies
```sh
$ npm install
```

### Run dev

```sh
# Development mode
$ npm run dev
```
> Open http://localhost:3000

### Build

```sh
# Production mode
$ npm run buld
```

### Run tests

```sh
$ npm run test
$ npm run test:watch
$ npm run test:coverage
```
